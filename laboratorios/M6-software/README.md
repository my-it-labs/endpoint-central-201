# M6 — Despliegue de software y configuración (2 h)

[← M5](../M5-automatizacion/README.md) · [Inicio del curso](../../README.md)

## Qué aprenderás

Desplegar software de forma controlada:

- Crear un **package** desde **template** (marketplace).
- Elegir **Install — Computer** vs **Install — User**.
- Configurar **policy**, **target**, filtros y exclusiones.
- Seguir el **estado de ejecución** hasta Success o Failed.

## Qué debes dominar al terminar M6

- Diferenciar template, package y deployment.
- Explicar por qué una tarea puede quedar en **Yet to Apply**.
- Verificar resultado en consola y, si aplica, en el cliente.
