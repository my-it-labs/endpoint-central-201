# M2 — Inventario y organización (4 h)

[← M1](../M1-arquitectura/README.md) · [Inicio del curso](../../README.md)

## Qué aprenderás

Endpoint Central mantiene **inventario** de software, equipos y hardware. En este módulo aprenderás **qué pregunta responde cada vista** y cómo forzar un escaneo desde un equipo concreto.

---

## Contexto (5 minutos)

| Vista | Responde a… |
|-------|-------------|
| **Computers** | ¿Qué sé de cada endpoint? (SO, último scan, estado…) |
| **Software** | ¿Qué aplicaciones hay en el parque? ¿En cuántos equipos? |
| **Hardware** | ¿Qué componentes (RAM, disco, CPU…) detecto? |

El inventario lo alimenta el **agente**; la consola agrega y filtra.

## Qué debes dominar al terminar M2

- Navegar las tres vistas de **Inventory** sin confundirlas.
- Explicar cuándo usar **Computers** vs **Software** vs **Hardware**.
- Lanzar **Asset Scan** y saber que el resultado tarda en reflejarse.
